# 📖 About

This repository contains my hands-on labs completed during the **iVolve Cloud DevOps Accelerator Program**.

The goal of this repository is to document my practical journey through modern Cloud and DevOps technologies by building real-world labs covering Build Tools, Docker, Kubernetes, CI/CD, GitOps, and Infrastructure Automation.

Each module contains practical exercises, Dockerfiles, Kubernetes manifests, automation scripts, and documentation demonstrating the concepts learned throughout the program.
